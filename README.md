# 🔴 Red Smart System

A hybrid system combining **Node.js server** and **C++ client** to simulate a real-time red team educational environment.

## 🚀 Features
- Real-time data communication (C++ → Node.js)
- Web-based dashboard
- Modular tasks (Task 1, Task 2, Task 3)

## 🛠️ Tech Stack
- Node.js
- Express.js
- C++
- HTML/CSS/JS

## ⚙️ Setup

### 1. Run Server
cd Master
npm install
node server.js

### 2. Run C++ Client
g++ main.cpp -o client.exe
./client.exe

## 📁 Project Structure
- Master → main server + client
- Task 1/2/3 → additional modules

## 📌 Notes
- Make sure both server and client are running
- Do not upload node_modules
